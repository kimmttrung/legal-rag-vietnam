# Ảnh hướng dẫn chạy trên Kaggle

Đặt các ảnh chụp màn hình vào đây, đúng tên file mà `README.md` (mục 7.1) tham chiếu:

| File ảnh | Nội dung cần chụp |
|----------|-------------------|
| `01_kaggle_gpu.png` | Settings → Accelerator chọn GPU T4 ×2 (hoặc P100) |
| `02_kaggle_secrets.png` | Add-ons → Secrets: thêm `QDRANT_URL`, `QDRANT_API_KEY` |
| `03_kaggle_upload_dataset.png` | Add Input → Upload dataset `R2AIStage1DATA.json` |
| `04_kaggle_input_path.png` | Copy file path trong panel Input → dán vào `INPUT_FILE` ở cell 5 |

> Chụp ảnh thật từ phiên Kaggle của bạn rồi lưu đè vào các tên file trên (giữ nguyên định dạng `.png`).
